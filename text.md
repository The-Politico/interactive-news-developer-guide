### Text alignment

```css
.center-align-text{
  text-align: center;
}

.left-align-text{
  text-align: left;
}

.right-align-text{
  text-align: right;
}
```

### Aligning content to the main well

```css
.content-fit{
  max-width: 600px;
  margin-left: auto;
  margin-right: auto;
}

.large{
  max-width: 800px;
  margin-left: auto;
  margin-right: auto;
}

.full-width{
  width: 100%;
  margin-left: auto;
  margin-right: auto;
}
```
