### Text alignment

```css
.text-align-center{
  text-align: center;
}

.text-align-center{
  text-align: left;
}

.text-align-center{
  text-align: right;
}
```

### Aligning content to the main well

```css
.content-fit{
  max-width: 550px;
  @include margin-auto;
}

.large{
  max-width: 800px;
  @include margin-auto;
}

.extra-large{
  max-width: 1200px;
  @include margin-auto;
}

.full-width{
  width: 100%;
  @include margin-auto;
}
```
